Wiresock doesn's support any global setting for exceptions from being tunneled. This scripts will add your exceptions to all files.
Just place the file in your Wiresock Profiles / Configs folder and add all the applications to the script. It will add all those applications to all profile / config files inside this folder.

Profile / config files normally to be found  ->   Windows harddrive/users/[URNAME]/Appdata/Local/WireSock Foundation/WireSock Secure Connect/Profiles/
