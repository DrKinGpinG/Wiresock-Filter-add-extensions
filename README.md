Wiresock doesn's support any global setting for exceptions from being tunneled. This scripts will add your exceptions to all files.
Just place the file in your Wiresock Profiles / Configs folder and use one connection .conf file, rename it to template.conf and add all the exceptions to it. It will add all those applications to all profile / config files inside this folder.

Profile / config files normally to be found  ->   In Windows "harddrive/users/[URNAME]/Appdata/Local/WireSock Foundation/WireSock Secure Connect/Profiles/"
